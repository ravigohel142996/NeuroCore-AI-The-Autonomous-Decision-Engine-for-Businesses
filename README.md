# NeuroCore AI — Autonomous Decision Intelligence System

> An enterprise-grade multi-agent AI system that forecasts revenue, predicts customer churn, detects cost anomalies, and generates strategic recommendations.

## Live Demo
🚀 [Deploy on Streamlit Cloud — https://neurocore-ai-the-autonomous-decision-engine-for-businesses.streamlit.app/] | [GitHub Repo](https://github.com/ravigohel142996/NeuroCore-AI-The-Autonomous-Decision-Engine-for-Businesses)

## What it does
- **Multi-agent architecture** — 4 specialized agents (Growth, Risk, Finance, Innovation) analyze business data independently
- **Consensus scoring** — agents vote on strategy recommendations using weighted consensus
- **Monte Carlo simulation** — probabilistic forecasting of future business outcomes
- **Interactive dashboard** — real-time decision intelligence visualization via Streamlit

## Tech stack
Python | Scikit-learn | Streamlit | Pandas | NumPy | Monte Carlo Simulation

## How to run
```bash
git clone https://github.com/ravigohel142996/NeuroCore-AI-The-Autonomous-Decision-Engine-for-Businesses
cd NeuroCore-AI-The-Autonomous-Decision-Engine-for-Businesses
pip install -r requirements.txt
streamlit run app.py
```
